# TheRecipesFinder

![Preview of TheRecipeFinder]()

TheRecipesFinder is simple recipe finder which find recipe using themealdb api which display the recipe image and catergory and instruction to cook the recipe and youtube link for guide.


## Overview
This project used  Rest api from TheMealDb to fetch content using javascript and display the content dynamicall.

## Features
* Search recipe by name
* Add recipe to favorite
* Show recipe video from youtube
* Display ingredient and measure needed
* Display Instruction

## Technologies USed
* HTML
* CSS
* JavaScript
* TheMealDb Api
